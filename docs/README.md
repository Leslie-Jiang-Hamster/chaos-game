1. `PRD.md`
   产品总览、核心体验、统一动作模型、聊天 UI、世界时钟、技术架构、实施路线。

3. `对话与记忆系统设计.md`
   消息写入、最近窗口、长期记忆、可见输入和记忆更新流程。
4. `回合结构设计.md`
   每轮每阶段的状态切换、`available_actions` 和接口开放规则。
5. `每轮游戏专属API说明.md`
   五轮游戏中的专属动作、参数、校验和结算用途。
