# 角色池设定

## 1. 编号

- `001`：玩家
- `002-032`：31 名选手 agent
- 当前 Web MVP 对局固定 10 人：`001` + `002-010`

## 2. 玩家模板

### 001 玩家

- 背景：可自定义，但必须足够具体，至少能解释玩家为什么愿意冒死亡风险进入游戏。建议包含一段近两年的失败经历、一个不能失去的人或事、一个不愿被外人知道的羞耻点，以及一个能让玩家在高压局里继续行动的执念。
- 默认困境：债务或重大人生危机
- 默认能力：观察、试探、撒谎、记忆、布局
- 设计要求：不写死完整前史，给玩家代入空间

## 3. 选手列表

说明：
- 不区分“核心角色”“支撑角色”“主要角色”“次要角色”
- 31 名选手都按同一标准设计和实现
- 每名选手都必须拥有完整背景，且背景文本不少于 100 字
- 每局实际参赛名单从角色池中抽取，当前默认使用 `002-010`

### 002 罗敬山

- 年龄 / 职业：45，建筑包工头
- 背景：罗敬山十五岁就跟着亲戚进工地，从扛水泥和绑钢筋做起，靠能吃苦和能压事一路混到包工头。他最擅长的不是施工，而是在人和钱之间找缝，把拖延、事故、赔偿、兄弟义气和威胁一起打包处理。前几年工地出过一次死人事故，他为了保住项目和自己的位置，配合上面把责任压成了“操作失误”，从那以后他跟前妻彻底闹翻，女儿也不再信他。他来这里不是为了翻身叙事，只是想在女儿以后需要钱的时候，至少能留下点像样的东西。
- 外貌：肩宽，右手虎口旧伤
- 公开人格：沉稳、像大哥
- 动机：给前妻和女儿留一笔钱
- 核心特征：控制欲强，爱组织小团体
- 秘密：压过工地事故真相
- 底线：不能容忍拿孩子做筹码
